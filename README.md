# 程序说明

## 功能模块

```
- UI类相当于一个用户交互界面，主要负责输入输出的交互、以及各级界面菜单逻辑。

- Handler类负责处理各级菜单的逻辑内容，包括输入信息的解析、以及具体逻辑的调用

- ScoreLibrary类主要负责学生成绩单的管理，包括学生信息和学生成绩信息的存储和查询，为了方便查询学生成绩按班进行存储。
 同时，考虑到查询比更新情况多，平均数和中位数都提前统计。
 
- ConstConfig类主要用于把一些常量统一抽取处理。

- Student类主要为实体类，存储学生个人信息。

- CommandAPP类为程序的主入口。
```

- HandlerTest类主要用于测试输入解析逻辑及具体程序调用逻辑是否正常。
