#程序说明
##功能模块
```
-UI类和Handler类主要用于和用户的交付，其中UI类只负责输入输出、Handler类只负责用户交互逻辑的处理。
-ScoreLibrary类主要负责学生成绩单的管理，包括学生信息和学生成绩信息的存储和查询，为了方便查询学生成绩按班进行存储。
 同时，考虑到查询比更新情况多，平均数和中位数都提前统计。
-ConstConfig类主要用于把一些常量同一抽取处理。
-Student类主要为实体类，存储学生个人信息。
-CommandAPP类为程序的主入口。
```
##测试模块
```
-ScoreLibraryTest类主要用于测试ScoreLibrary添加学生、添加成绩、查询成绩、统计、以及加分功能的功能是否正常。
-StudentTest类主要用于测试Student类equal方法。
-由于UI类和Handler类主要是用户交互相关功能，同时，相应的逻辑和功能也较为简单，所以直接进行功能测试，没有使用单元测试。
 具体测试内容如：result.png
```
